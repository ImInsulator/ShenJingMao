# ShenJingMao
it's (maybe) the most difficult version in the history of "surround Nervous Cat".

(也许是)史上最难的围住神经猫。
网络上大多数围住神经猫基于贪心算法，因此很容易用“挖陷阱”的方式将其捉住。
这里基于Qt on Android实现了具有三个不同难度模式的围住神经猫，第二个难度即采用了贪心法，第三个难度基于动态路径规划与博弈论实现，因此难度有很大提升。
